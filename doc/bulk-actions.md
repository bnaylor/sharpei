# Bulk Actions

Sharpei allows you to manage multiple tasks simultaneously using **Bulk Edit** mode. This is useful for reorganizing your list, changing priorities, or cleaning up many tasks at once.

## Entering Bulk Edit Mode

To start managing multiple tasks:
1.  Click the **Bulk Edit** button in the toolbar.
2.  The standard completion checkboxes will be replaced by **selection checkboxes** on the left side of each task.
3.  A **Bulk Action Bar** will appear at the bottom of the screen as soon as you select at least one task.

## Selecting Tasks

-   **Individual Selection**: Click the checkbox on the left of any task.
-   **Range Selection**: Click the checkbox of the first task, then hold **Shift** and click the checkbox of another task to select everything in between.
-   **Select All**: Click the "Select all" icon (square) in the toolbar to select every currently visible task.

## Available Bulk Actions

Once you have selected one or more tasks, you can use the Bulk Action Bar to:

| Action | Description |
|--------|-------------|
| **Priority** | Change the priority (High, Normal, Low) for all selected tasks. |
| **Move to...** | Move all selected tasks to a specific category (or remove them from a category). |
| **Mark Done** | Mark all selected tasks as completed. |
| **Mark Pending** | Unmark all selected tasks (mark as not completed). |
| **Archive** | Move all selected completed tasks to the archive. |
| **Delete** | Permanently remove all selected tasks. A confirmation dialog will appear. |

## Exiting Bulk Edit Mode

Click **Exit Bulk Edit** in the toolbar or click **Cancel** in the Bulk Action Bar to return to normal mode. All current selections will be cleared.
